# Angular 
Angular seed project using the latest version of Angular, Typescript, SystemJS, Gulp, BrowserSync...

With live reload.

## Versions used

* Angular 5.0
* Typescript 2.5
* Gulp 4
* SystemJS 0.20

## How to use it
```
$ git clone git@github.com:RilRil/Angular.git
$ cd Angular/
$ npm install
$ npm start
```
