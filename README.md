# Angular 
Angular seed project using the latest version of Angular, Typescript, SystemJS, Gulp, BrowserSync, Bootstrap, ...

## Versions used

* Angular 5.1
* Typescript 2.6
* Gulp 4
* SystemJS 0.20
* Bootstrap 4

## How to use it
```
$ git clone git@github.com:RilRil/Angular.git
$ cd Angular/
$ npm install
$ npm start
```
